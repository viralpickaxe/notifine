notifine

[noh-tih-FH-hyne]

What is this?

notifine, is a lightweight but powerful jQuery notification system. Jazz up your site with these good looking, responsive, growl like notifications.

Why should I use this?

notifine is very easy to use and has lots of great features including;

  • Default notifications are responsive and good looking
  • The look, positioning and behaviour of the notifications are fully customizable
  • Event callbacks for many different actions
  • Simple but powerful API for creating, detecting and destroying notifications easily

Will it slow me down?

No of course not! notifine was created so that you don't have to worry about having to create a whole seperate system just to give the user feedback. Our API is super easy to use and implement, and once you get the hang of it you can take advantage of all the more advanced features that notifine has.

notifine's js is 3kb compressed (4kb uncompressed), and it's default css is 2kb compressed & uncompressed (scss)

Documentation


Basic Notifications

These are the most basic type of notification that you can create. notifine comes with 6 different types/colours of notifications. By default the notification will dismiss after 3000ms (or until clicked). If a type is not specified it will create a grey type notification.

$(...).click(function(){
  var notification = Notifine.create(
    {
      content : {
        "title" : "This is a notification",
        "body" : "This is the body content of a notification."
      },
      type : "purple" // Works for any colour class added
    }
  );
});

Notifine.create() also returns a notification data object. Infomation like the id might be important to keep track of incase you want to do things like dismissing a notification from the API.

{
  id : "0x7r9",
  content : {
    title : "This is a notification",
    body : "This is the body content of a notification."
  },
  type : "orange",
  time : 0,
  events : {}
}

Timing Notifications

If you do not define a "time" item in the Notifine.create() object, the notification will just self dismiss after 3000ms. Simply set time equal to the amount of miliseconds you want before it dismisses. By setting the time to 0, the notification will not dismiss until it is clicked.

$(...).click(function(){
  var notification = Notifine.create(
    {
      content : {
        "title" : "This is a notification",
        "body" : "This is the body content of a notification."
      },
      type : "purple",
      time : 5000
    }
  );
});

Event Listner Notifications

One of the awesome upsides to using notifine, is that you have access to a few event listeners for individual notifications.


Dismiss/Destroy Notifications

As well as using timings or onclick to dismiss notifications, you can use a function. Just call Notifine.dismiss(notificationid) with a notificationid (the 5 character long string that gets returned from Notifine.create()).

Notifine.dismiss("0x7r9");
// Does the same thing as Notifine.destroy("0x7r9");
