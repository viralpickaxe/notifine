notifine

[noh-tih-FH-hyne]

What is this?

notifine, is a lightweight but powerful jQuery notification system. Jazz up your site with these good looking, responsive, growl like notifications.

Why should I use this?

notifine is very easy to use and has lots of great features including;

  • Default notifications are responsive and good looking
  • The look, positioning and behaviour of the notifications are fully customizable
  • Event callbacks for many different actions
  • Simple but powerful API for creating, detecting and destroying notifications easily

Will it slow me down?

No of course not! notifine was created so that you don't have to worry about having to create a whole seperate system just to give the user feedback. Our API is super easy to use and implement, and once you get the hang of it you can take advantage of all the more advanced features that notifine has.

notifine's js is 3kb compressed (4kb uncompressed), and it's default css is 2kb compressed & uncompressed (scss)
